# NgFirebaseUpload
This project was created to study and implement upload files, functionality it's present in many common web applications. We can use that to upload files (documents and images) and storage our files on the Firestore. 

## Running the project
The first step is create Firebase project and insert configs into environment file. Then execute the following command:

``` ng serve ```
